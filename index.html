Introduction to Python

Python is a high-level, interpreted programming language known for its simplicity and readability. Created by Guido van Rossum and first released in 1991, Python has become one of the most popular programming languages worldwide.

Key features of Python include:

  • Easy-to-learn syntax that emphasizes readability
  • Support for multiple programming paradigms (object-oriented, functional, procedural)
  • Comprehensive standard library
  • Cross-platform compatibility
  • Large and active community

Your First Python Program

print("Hello, World!")

Python Syntax

Python syntax is designed to be intuitive and easy to read. Unlike many other languages, Python uses indentation to define code blocks rather than curly braces or keywords.

Basic syntax rules:

  • Statements end at the end of the line (no semicolons needed)
  • Code blocks are defined by indentation (typically 4 spaces)
  • Comments start with #
  • Variables are created when first assigned

Syntax Example

 # This is a comment
x = 5  # Creating a variable
if x > 0:
    print("Positive number")
else:
    print("Non-positive number")

Variables and Data Types

Variables in Python are used to store data values. Unlike some other languages, Python is dynamically typed, meaning you don't need to declare variable types explicitly.

Basic data types in Python:

  • Numbers: int, float, complex
  • Strings: str
  • Boolean: bool (True/False)
  • Sequences: list, tuple, range
  • Mappings: dict
  • Sets: set, frozenset

Variables Example

# Integer
age = 25

# Float
price = 19.99

# String
name = "Alice"

# Boolean
is_active = True

# List
colors = ["red", "green", "blue"]

# Dictionary
person = {"name": "Bob", "age": 30}

Operators

Python has various types of operators for performing operations on variables and values.

Operator categories:

  • Arithmetic: +, -, *, /, %, **, //
  • Comparison: ==, !=, >, <, >=, <=
  • Logical: and, or, not
  • Assignment: =, +=, -=, *=, etc.
  • Identity: is, is not
  • Membership: in, not in

Operators Example

# Arithmetic
x = 10 + 5  # 15
y = 10 * 2  # 20

# Comparison
print(10 > 5)  # True

# Logical
if x > 0 and y > 0:
    print("Both positive")

# Membership
colors = ["red", "green", "blue"]
print("red" in colors)  # True

Conditional Statements

Conditional statements allow you to execute different blocks of code based on certain conditions.

Types of conditional statements:

  • if - executes code if condition is true
  • elif - checks another condition if previous conditions were false
  • else - executes code if all conditions are false

Conditionals Example

age = 18

if age < 13:
    print("Child")
elif age < 18:
    print("Teenager")
else:
    print("Adult")

# Ternary operator
status = "Adult" if age >= 18 else "Minor"

Loops

Loops allow you to execute a block of code repeatedly.

Types of loops in Python:

  • for - iterates over a sequence
  • while - executes while condition is true
  • Loop control: break, continue, pass

Loops Example

# For loop
for i in range(5):
    print(i)  # Prints 0 to 4

# While loop
count = 0
while count < 5:
    print(count)
    count += 1

# Loop control
for num in range(10):
    if num == 5:
        break  # Exit loop
    if num % 2 == 0:
        continue  # Skip even numbers
    print(num)

Functions

Functions are reusable blocks of code that perform a specific task. They help organize code and avoid repetition.

Key concepts:

  • Defined with def keyword
  • Can accept parameters
  • Can return values using return
  • Can have default arguments
  • Support variable-length arguments (*args, **kwargs)

Functions Example

# Basic function
def greet(name):
    return f"Hello, {name}!"

print(greet("Alice"))  # Hello, Alice!

# Function with default argument
def power(base, exponent=2):
    return base ** exponent

print(power(3))      # 9
print(power(3, 3))   # 27

# Variable-length arguments
def sum_all(*numbers):
    total = 0
    for num in numbers:
        total += num
    return total

print(sum_all(1, 2, 3))  # 6

Lists

Lists are ordered, mutable collections of items. They are one of Python's most versatile data structures.

List characteristics:

  • Created with square brackets []
  • Can contain items of different types
  • Support indexing and slicing
  • Many built-in methods: append(), remove(), sort(), etc.

Lists Example

# Creating a list
fruits = ["apple", "banana", "cherry"]

# Accessing elements
print(fruits[0])   # apple
print(fruits[-1])  # cherry (last element)

# Slicing
print(fruits[1:3])  # ['banana', 'cherry']

# Modifying
fruits.append("orange")
fruits[1] = "blueberry"

# List methods
numbers = [3, 1, 4, 1, 5]
numbers.sort()
print(numbers)  # [1, 1, 3, 4, 5]

# List comprehension
squares = [x**2 for x in range(5)]
print(squares)  # [0, 1, 4, 9, 16]

Tuples

Tuples are similar to lists but are immutable (cannot be changed after creation). They are often used for fixed collections of items.

Tuple characteristics:

  • Created with parentheses ()
  • Immutable (cannot add, remove, or change items)
  • Faster than lists for fixed data
  • Can be used as dictionary keys (unlike lists)

Tuples Example

# Creating a tuple
colors = ("red", "green", "blue")

# Accessing elements
print(colors[0])   # red
print(colors[1:3]) # ('green', 'blue')

# Tuple unpacking
x, y, z = colors
print(y)  # green

# Single-element tuple (note the comma)
single = (42,)

# Tuple as dictionary key
locations = {
    (35.6895, 139.6917): "Tokyo",
    (40.7128, -74.0060): "New York"
}

Dictionaries

Dictionaries are unordered collections of key-value pairs. They are optimized for fast lookup of values based on keys.

Dictionary characteristics:

  • Created with curly braces {}
  • Keys must be immutable (strings, numbers, tuples)
  • Values can be any type
  • No duplicate keys allowed
  • Many useful methods: keys(), values(), items(), etc.

Dictionaries Example

# Creating a dictionary
person = {
    "name": "Alice",
    "age": 25,
    "city": "New York"
}

# Accessing values
print(person["name"])  # Alice
print(person.get("age"))  # 25

# Adding/modifying
person["email"] = "alice@example.com"
person["age"] = 26

# Dictionary methods
print(person.keys())   # dict_keys(['name', 'age', 'city', 'email'])
print(person.values()) # dict_values(['Alice', 26, 'New York', 'alice@example.com'])

# Dictionary comprehension
squares = {x: x*x for x in range(5)}
print(squares)  # {0: 0, 1: 1, 2: 4, 3: 9, 4: 16}

Classes and Objects

Python is an object-oriented programming language. Classes are blueprints for creating objects, which bundle data and functionality together.

Key concepts:

  • Defined with class keyword
  • __init__ method for initialization
  • self refers to the instance
  • Supports inheritance and polymorphism
  • Special methods (__str__, __len__, etc.)

Classes Example

# Simple class
class Dog:
    def __init__(self, name, age):
        self.name = name
        self.age = age
    
    def bark(self):
        return "Woof!"

# Creating an object
my_dog = Dog("Buddy", 3)
print(my_dog.name)    # Buddy
print(my_dog.bark())  # Woof!

# Inheritance
class Bulldog(Dog):
    def __init__(self, name, age, weight):
        super().__init__(name, age)
        self.weight = weight
    
    def bark(self):
        return "Woof! Woof!"

bulldog = Bulldog("Max", 2, 25)
print(bulldog.bark())  # Woof! Woof!

Modules

Modules are Python files containing reusable code. They help organize code into logical units and promote code reuse.

Key concepts:

  • Import modules using import
  • Access module contents with dot notation
  • Can import specific items with from...import
  • Standard library contains many useful modules
  • Can create your own modules

Modules Example

# Importing a module
import math

print(math.sqrt(16))  # 4.0

# Importing specific functions
from random import randint

print(randint(1, 10))  # Random number between 1-10
